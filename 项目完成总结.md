## 🚀 如何获取APK

### 方法1：GitHub在线构建（推荐）
1. 登录GitHub.com
2. 创建新仓库
3. 上传项目文件
4. GitHub Actions自动构建
5. 下载生成的APK

**优点：** 无需下载任何软件，免费，快速

### 方法2：本地构建
1. 安装Android Studio
2. 打开项目
3. 点击Build → Build APK
4. 在app/build/outputs/apk/debug/获取APK

**优点：** 可以自定义修改代码
